## Description
This folder contains all of the experiments I do to explore the JavaScript, CSS, and HTML worlds. Projects here are guided by the Codecademy full-stack path, a course that has allowed me to put into practice different concepts along with my own ideas and creativity. 
With this folder, I intend to not only expand my learning and programming skills, but also to keep track of how my projects change along with me.

* Projects that get more complex everytime. Hopefully not more complex to understand, but only with more implementations.
* The beginning and end of my personal coding journey.
